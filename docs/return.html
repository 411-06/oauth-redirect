<!DOCTYPE html>

<html lang="en">

<head>

  <meta charset="UTF-8" />

  <title>OAuth Redirect</title>

  <script>

    // 認可コードをURLから抜き出してlocalStorageに保存

    const params = new URLSearchParams(window.location.search);

    const code = params.get("code");

    if (code) {

      localStorage.setItem("google_auth_code", code);

    }

    // 自動でウィンドウを閉じる or リダイレクト

    window.close(); // これでポップアップ閉じられる（WebGL対応）

  </script>

</head>

<body>

  <p>認証が完了しました。画面を閉じてください。</p>

</body>

</html>
